Before you start please structure your data in the following manner

Username Weekday Start End Tendency Note
  • Start, End and Note are optional. All other fields are required.
  • Weekday has to be written like "Monday", etc.
  • If Available is "Yes" it will be set to True, otherwise False
  • Start and end have to be in the format "HH:mm"
    • Tendency only contains a number:
    • 0 - No preference
    • 1 - Early shift
    • 2 - Midday shift
    • 3 - Late shift
    • 4 - Night shift
