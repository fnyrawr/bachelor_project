Before you start

Please structure your data in the following manner:

Date Start End Break Department Username Highlight Note

Note and Username and Highlight are optional. All other fields are required.

Date has to be formatted "YYYY-MM-DD"

Start, End and Break have to be formatted "HH:mm"

Department and Username have to be exact to the entries in the database!

"Yes" in Highlight column will set the Highlight value to True, otherwise False

Qualifications are currently not importable with shifts
