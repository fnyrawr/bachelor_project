Before you start

Please structure your data in the following manner:

Username Date Start End Tendency Note

Start, End and Note are optional. All other fields are required.

Date has to be in ISO format "YYYY-MM-DD"

If Available is "Yes" it will be set to True, otherwise False

Start and end have to be in the format "HH:mm"

    Tendency only contains a number:
  • 0 - No preference
  • 1 - Early shift
  • 2 - Midday shift
  • 3 - Late shift
  • 4 - Night shift
